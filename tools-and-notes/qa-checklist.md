# ✅ QA Testing Checklist

## Before Testing
- Understand requirements
- Clarify expected behavior
- Prepare test data

## Functional Testing
- Happy path scenarios
- Negative testing
- Edge cases

## Bug Reporting
- Clear steps to reproduce
- Expected vs Actual result
- Add screenshots or logs

## API Testing
- Check status codes
- Validate JSON responses
- Test error handling
